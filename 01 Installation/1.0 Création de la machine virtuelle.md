La phase d’installation va vous permettre de préparer une VM, installer l’OS et créer un utilisateur non-root pour suivre ce TP.

Pour ce TP, vous allez utiliser une VM virtual box![[1.0 Environnement TP labo.png]]
# Récupérez l'ISO
Télécharger l'ISO Alpine Linux `alpine-extended-3.22.2-x86_64.iso` depuis:
SIONAS -> SIO1 -> SIO1 SISR -> TP AlpineLinux 

Ou téléchargez la depuis: le [site officiel](https://dl-cdn.alpinelinux.org/alpine/v3.22/releases/x86_64/alpine-standard-3.22.2-x86_64.iso)

# Créez une nouvelle VM

## **1. Name and Operating System**

- **Nom :** `Alpine 3.22.2`
- **Folder :** garder le dossier par défaut ou choisir `~/VirtualBox VMs`
- **ISO Image :** sélectionner l’ISO `alpine-extended-3.22.2-x86_64.iso`
- **Edition :** _(laisser vide)_
- **Type :** `Linux`
- **Subtype :** `Other Linux`
- **Version :** `Other Linux (64-bit)`
- **Skip Unattended Installation :** **cocher** (important)

![[1.0 Environnement TP labo-1.png]]

## Hardware
-  **CPU :** 1 vCPU  
-  **RAM :** 512 Mo  

![[1.0 Environnement TP labo-2.png]]

## **Hard Disk**

### **Disque virtuel :**

-  **Create a virtual hard disk now**
-  **Format :** VDI
-  **Storage :** Dynamically allocated
-  **Size :** 1 Go

![[1.0 Environnement TP labo-3.png]]
# Configuration de votre VM
## Configuration réseau Adaptateur 1
- Cochez "activez l'interface réseau"
 - Mode d'accès réseau : RéseauNAT (par défault `NatNetwork`)
![[1.0 Environnement TP labo-4.png]]
