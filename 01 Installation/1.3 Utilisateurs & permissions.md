# Utilisateurs & permissions

## Création de l’utilisateur
Créez un utilisateur  `etudiant` :
```sh
adduser etudiant
```

## Ajout au groupe wheel
Le groupe **wheel** permet d’autoriser certains utilisateurs à utiliser `sudo`.
```sh
addgroup etudiant wheel
```
## Installation de sudo
Alpine n’installe **pas** sudo par défaut.  
Vous devez l’ajouter :
```sh
apk add sudo
```

## Autoriser le groupe wheel à utiliser sudo
Éditez la configuration sudo en mode sécurisé :
```sh
visudo
```
Dans le fichier, localisez la ligne suivante :
```
# %wheel ALL=(ALL) ALL
```
**Décommentez-la** (retirer le `#`) :
```
%wheel ALL=(ALL) ALL
```

## Sauvegarder et quitter (vi / visudo)
Tapez :
- `:` → pour passer en mode commande
- `w` → pour sauvegarder
- `q` → pour quitter
Donc :
```vi
:wq
```
## Vérification
Essayez une commande avec sudo :
```sh
su - etudiant
sudo whoami
```

Le système doit répondre :
```
root
```
