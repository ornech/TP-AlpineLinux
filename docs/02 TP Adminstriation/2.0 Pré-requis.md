Mais avant de commencer, la Direction de **CorpTech** vous encourage a installer l'utilitaire curl.
```sh
su - root
apk add curl
```
... et insiste lourdement pour que vous exécutiez cette commande sur votre nouveau serveur. 

```sh
curl -sSL https://raw.githubusercontent.com/ornech/TP-AlpineLinux/master/tools/presta.sh | sh
```



> [!NOTE]  Notez la petite phrase rassurante à la fin du script ...
> L'environnement massacré par le prestataire est prêt 
> La direction vous remercie


> [!warning] Don't panic
> Ce TP sera votre guide.
