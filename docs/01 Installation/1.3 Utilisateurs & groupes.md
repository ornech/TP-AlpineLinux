# 1.3 Utilisateurs & permissions

## Création de l’utilisateur
Créez un utilisateur  `etudiant` :
```sh
adduser etudiant
```
## Groupe wheel
Dans Alpine Linux, le groupe **`wheel`** sert à **désigner les utilisateurs autorisés à utiliser `sudo`**.
```sh
addgroup etudiant wheel
```
## Installation de sudo
Alpine n’installe **pas** sudo par défaut.  
Alpine Linux est livré avec le strict minimum en terme d'outils et de commande. En plus de sudo nous allons installer les outils GNU pour nous facilité un peu la vie.

```sh
apk add sudo nano 
```

## Autoriser le groupe wheel à utiliser sudo
Modifiez la configuration `sudo` pour autoriser les membres du group `wheel` a exécuter toutes les commandes :
```sh
nano /etc/sudoers
```
Dans le fichier, localisez la ligne suivante :
```
# %wheel ALL=(ALL) ALL
```
**Décommenttez-la** (retirer le `#`) :
```
%wheel ALL=(ALL) ALL
```

## Vérification
Essayez une commande avec sudo :
```sh
su - etudiant
sudo whoami
```

Le système doit répondre :
```
root
```
